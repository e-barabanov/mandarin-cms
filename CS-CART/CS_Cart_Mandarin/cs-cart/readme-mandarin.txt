* Модуль оплаты через платежный шлюз МандаринБанк
* Совместим с версией CS-cart 4.3.x (4.2.х не поддерживается)


УСТАНОВКА
====================

*Автоматическая:*

1. Перейти в админ панели в раздел "Модули - Управление модулями", нажать на кнопку установки нового модуля в правом верхнем углу
2. Выбрать файл на компьютере, нажать "Установить". (права на папки app/addons, app/payments должны быть 777)
3. В случае успешной установки, активировать модуль. (Он появится в списке доступных модулей)

*Ручная:*

1. Распаковать архив с модулем
2. Скопировать вручную файлы на сервер в корневую папку
3. Найти модуль в списке доступных модулей, активировать его.

НАСТРОЙКА
===================

1. Перейти в раздел "Администрирование - Способы оплаты", нажать добавить способ оплаты ("+")
2. Название модуля "МандаринБанк", выбрать процессор из списка "mandarinbank", Категория оплаты "Интернет платежи"
3. После выбора станет доступна закладка Настроек модуля.
4. Заполняем поля 

Индефикатор кассы: ХХХ
Секретный ключ: ХХХХХХХХ

5. Сохраняем, модуль готов к работе

callbackURL: %домен%/?dispatch=payment_notification.notify&payment=mandarinbank
returnURL: %домен%/?dispatch=payment_notification.success&payment=mandarinbank
