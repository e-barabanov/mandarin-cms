Установка
1. Копируем файлы из папки с плагином на сервер с установленной CMS phpShop в корень сайта.
2. Переходим по ссылке %ваш_домен%/install-mandarin.php Должна появится надпись "delete /install-mandarin.php".
3. Удаляем файл /install-mandarin.php
4. Заходим в админ панель %ваш_домен%/phpshop/admpanel/ Далее "Веб-сайт" > "Страницы" > там будет плюсик справа вверху (Добавить страницу).
5. Вводите заголовок, URL Ссылка - /page/result.html, опции вывода: вывод в каталоге
6. Нажимаем кнопку "создать и редактировать"
7. В правом верхнем углу ищем "Модули ^", нажимаем > Управление модулями. Ищем 
"MandarinBank 1.0
Прием оплат через MandarinBank", наводим на него появляется шестерёнка, нажимаем на неё > Включить.
8. Далее конфигурируем. Нажимаем на "Модули ^" > Приём оплат через MandarinBank > вводим Merchant ID > вводим Merchant Secret > описание оплаты можно не трогать > В правом верхнем углу кнопка сохранить.

Покупка
1. Заносим любой товар в корзину.
2. Нажимаем кнопку оформить заказ.
3. В корзине выбираем способ оплаты MandarinBank
4. На странице оплаты нажимаем кнопку оплатить

callbackURL http://%ваш_домен%/payment-mandarin.php
resultURL http://%ваш_домен%/page/result.html
